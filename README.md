# Origin
Data-driven analysis of the 2008 financial crisis, examining market drawdowns, sector-level impacts, volatility dynamics, and recovery timelines using financial and macroeconomic data.

# Research Questions
1. How much did major indices fall during 2008?
2. How long did recovery take?
3. Which industries recovered first/faster?
4. Which sectors were most impacted?
5. How did volatility behave before, during, and after the crash?
6. Benchmark against unemployment rates
7. Benchmark against Debt ratio
